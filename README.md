# serverprice

## Usage
1. check all available region for cloud platform

```bash
> python -m serverprice list_region aliyun
```

2. get price for server by cpu core count and memory

```bash
> python -m serverprice get aliyun -c 1 -m 1 -r cn-qingdao
```
